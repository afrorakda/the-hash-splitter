# The Hash Splitter (Fridge Combo #119)

A precision tool for social media managers and bloggers to instantly clean, deduplicate, and format hashtags for X (Twitter) and Instagram.

## Concept: "Polish Your Reach"
Don't let messy tags slow down your posting. Whether you're pulling keywords from a draft or reusing tags from an old post, this tool ensures your hashtags are unique, correctly formatted, and ready for maximum impact.

## Core Features
- **Smart Extraction**: Automatically strips commas, dots, redundant `#` symbols, and weird spaces to find the core keywords.
- **Deduplication (Unique)**: Automatically removes duplicate tags to save character space and keep your posts professional.
- **Dual Formatting**:
    - **FOR X (SPACE)**: Formats tags horizontally with spaces for high-speed Twitter posting.
    - **FOR INSTA (NEWLINE)**: Formats tags vertically for clean Instagram captions and comments.
- **Character Counter**: Real-time count to help you stay within platform limits.

## Design Philosophy
- **1 Tool / 1 Action**: Paste raw text, pick your platform, and you're done.
- **Fridge Standard**: Clean #333 UI with 15px rounded corners for a "professional tool" feel.
- **Zero Friction**: One-tap formatting and automatic clipboard copy.

## Technical Standards
- **Privacy-First**: No data leaves your browser. Safe for confidential campaign tags.
- **Mobile First**: Optimized for quick operation on smartphones between shoots or commutes.
- **iOS Guard**: Prevents system pop-ups from interrupting your workflow.
